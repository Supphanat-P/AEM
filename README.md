# AEM Macro

โปรเจกต์นี้เป็นสคริปต์ AutoHotkey (`AEM2.ahk`) สำหรับบันทึกและเล่น Macro ในเกม พร้อมโหมด Auto Farm ที่ใช้การค้นหาภาพหน้าจอส่วนต่าง ๆ เช่น `start.png`, `here.png`, `restage.png`, `cancel.png` เพื่อควบคุมลูปเกมอัตโนมัติ

## คุณสมบัติหลัก

- บันทึกจุดคลิกและปุ่มคีย์แบบ Relative ต่อหน้าต่างเป้าหมาย
- บันทึกค่าเวลา delay ระหว่างการกระทำ
- เล่น Macro ซ้ำได้โดยอัตโนมัติ
- รองรับการโหลดและบันทึก Macro เป็นไฟล์ `.ini`
- ควบคุมโดยปุ่มฟังก์ชันและ GUI ง่าย

## การเตรียมใช้งาน

1. ติดตั้ง AutoHotkey: https://www.autohotkey.com
2. เปิดโฟลเดอร์ `AEM` และตรวจสอบว่าไฟล์ภาพสกรีนช็อตมีอยู่:
   - `start.png`
   - `here.png`
   - `restage.png`
   - `cancel.png`
3. เปิดไฟล์ `AEM2.ahk` ด้วย AutoHotkey หรือดับเบิลคลิกเพื่อรันสคริปต์

## วิธีใช้งาน

### เริ่มบันทึก Macro

1. เปิดเกมและนำหน้าต่างเกมมาที่หน้าจอหลัก
2. กดปุ่ม `F1` หรือคลิกปุ่ม `Record` ใน GUI
3. ทำการคลิกและกดคีย์ตามลำดับที่ต้องการในเกม
4. กดปุ่ม `F2` หรือคลิกปุ่ม `Stop Rec` เพื่อหยุดการบันทึก

### บันทึก Macro ลงไฟล์

1. คลิกปุ่ม `Save Macro`
2. เลือกตำแหน่งและตั้งชื่อไฟล์ `.ini`
3. Macro จะถูกบันทึกพร้อมข้อมูลจำนวนขั้นตอน และหน้าต่างเป้าหมายที่จับได้

### โหลด Macro จากไฟล์

1. คลิกปุ่ม `Load Macro`
2. เลือกไฟล์ `.ini` ที่บันทึกไว้
3. สคริปต์จะแสดงว่าโหลด Macro เรียบร้อยและเตรียมพร้อมใช้งาน

### เริ่ม Auto Farm

1. กดปุ่ม `F3` หรือคลิกปุ่ม `Start` ใน GUI
2. สคริปต์จะทำงานดังนี้:
   - ค้นหา `start.png` และคลิกปุ่มเริ่มเกม
   - เล่น Macro ที่บันทึกไว้
   - ค้นหา `here.png` และคลิกเพื่อต่อ
   - ค้นหา `restage.png` แล้วคลิกเพื่อรันรอบใหม่
   - รอจนกว่าปุ่ม `cancel.png` จะหายไปเพื่อหมายถึงการโหลดเสร็จ

### หยุดการทำงาน

- กดปุ่ม `F4` หรือคลิกปุ่ม `Stop` ใน GUI
- หากต้องการปิดโปรแกรมทั้งหมด ให้กด `j`

## ปุ่มลัดสำคัญ

- `F1` - เริ่มบันทึก Macro
- `F2` - หยุดบันทึก Macro
- `F3` - เริ่มสคริปต์ Auto Farm
- `F4` - หยุดสคริปต์
- `j` - ปิดโปรแกรม

## หมายเหตุสำคัญ

- ให้คลิกที่หน้าต่างเกมก่อนกด `F1` เพื่อให้สคริปต์จับตำแหน่งถูกต้อง
- หากหน้าต่างเกมถูกย้ายหลังจากบันทึก Macro สคริปต์จะยังพยายามใช้งานตำแหน่ง Relatve ต่อหน้าต่าง
- ภาพ `start.png`, `here.png`, `restage.png`, `cancel.png` ต้องชัดเจนและตรงกับหน้าจอเกมจริง
- หากไม่พบภาพในเวลาที่กำหนด สคริปต์จะใช้การคลิกสำรองในตำแหน่งค่าประมาณ

## โฟลเดอร์สำรอง

- `saved/` - โฟลเดอร์สำหรับไฟล์ `.ini` ที่ Save Macro หากต้องการจัดเก็บเพิ่มเติม

## แก้ไขและปรับแต่ง

- หากต้องการเพิ่มคีย์ที่บันทึกได้ ให้แก้ค่าตัวแปร `RecordKeys` ใน `AEM2.ahk`
- หากต้องการปรับความช้าของเมาส์หรือการคลิก ให้แก้ `SetDefaultMouseSpeed` และ `SetMouseDelay`
- หากต้องการเปลี่ยนเวลารอการค้นหาภาพ ให้แก้ค่าตัวเลขในฟังก์ชัน `FindImage`

## การเรียกใช้งาน

- ดับเบิลคลิก `AEM2.ahk`
- หรือคลิกขวา > Run Script

## ปัญหาที่อาจเจอ

- หากสคริปต์ไม่ทำงาน ให้ตรวจสอบว่า AutoHotkey ติดตั้งแล้วและเวอร์ชันรองรับ
- หาก GUI ไม่แสดง ให้ตรวจสอบว่าไม่มีสคริปต์ AHK ตัวอื่นรันทับ
- หากไม่พบปุ่มในเกม ให้เก็บภาพปุ่มใหม่ด้วยความละเอียดและสภาพแสงที่ชัดเจน
